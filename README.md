# RE-Man

RE-Man is a python script for CTF players.

## Installation
```
git clone https://github.com/silica47/re-man
cd re-man
chmod +x requirements.sh
sudo ./requirements.sh
```

## Usage

```
cd re-man
cp <path-to-binary> .
python3 re-man.py -f <name-of-the-binary> -i <optional-arguments> 
```
